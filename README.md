# Flutter — Lince Tech Academy

Exercícios práticos de Flutter do treinamento da Lince Tech Academy.

Cada arquivo é um app completo e independente: basta usá-lo como `lib/main.dart`
de um projeto Flutter e executar com `flutter run`.

| Pasta | Arquivo | Exercício |
| --- | --- | --- |
| `apf1_funcoes` | `ap1.dart` | Cores aleatórias |
| | `ap2.dart` | Opção aleatória |
| | `ap3.dart` | Refatoração |
| `apf2_funcoes` | `ap1.dart` | Retângulos coloridos |
| | `ap2.dart` | Quadrados empilhados |
| | `ap3.dart` | Formatos dinâmicos |
| `apf3_funcoes` | `ap1.dart` | Formulário básico |
| `apf4_funcoes` | `ap1.dart` | Lista de cores (navegação entre telas) |

## Prompts de IA

A pasta `ia_prompts` reúne as atividades de uso de IA como apoio nos temas de
Flutter: introdução e widgets, formulários, rotas e widgets.

Os exercícios de Dart (sem Flutter) ficam no repositório
[Lince_Academy](https://github.com/joaorodrigues351/Lince_Academy).
